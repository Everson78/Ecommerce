# Projeto Banco de Dados Ecommerce

Projeto desenvolvido como parte do desafio de Banco de Dados da DIO.

## Objetivo

Construir um banco de dados relacional para um cenário de ecommerce, aplicando conceitos de modelagem, criação de tabelas e consultas SQL.

## Tecnologias utilizadas

- MySQL
- MySQL Workbench

## Conceitos aplicados

- Modelagem relacional
- Chaves primárias (PK)
- Chaves estrangeiras (FK)
- Relacionamentos 1:N
- Relacionamentos N:N
- Constraints
- Queries SQL

## Estrutura do banco

Principais entidades:

- Cliente
- Cliente PF/PJ
- Produto
- Pedido
- Pagamento
- Entrega
- Estoque
- Fornecedor
- Terceiro Vendedor

## Consultas realizadas

Foram criadas queries utilizando:

- SELECT
- WHERE
- JOIN
- GROUP BY
- HAVING
- ORDER BY
- Funções de agregação
